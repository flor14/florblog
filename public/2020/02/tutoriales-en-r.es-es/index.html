Tutoriales en R

20 February 2020 in Tutoriales

Tutoriales

Existen dos formas de realizar tutoriales en R, con el framework gratuito que creo Ines Montani y/o el paquete learnR.

El post en el que recientemente colaboré junto a Ted Laderas, quien desarrollo el paquete decampr y Jessica X para el blog de RStudio Education comenta algunas diferencias entre ambas formas de realizar tutoriales.

Puedes encontrar una lista de todos los tutoriales realizados con este framework aquí.

En este post voy a generar una guía rápida sobre como realizar desde 0 un tutorial con el framework de Ines Montani. ¿Sos un profresor de estadística y querés dejar tu tutorial de online? Cómo mencioné este framework no se usa desde R por lo qur puede presentar ciertas dificultades para alguien no familiarizado.

Warning! Un tutorial no es lo mismo que un curso, podés leer más sobre esto aquí.

Receta para tu tutorial con el framework de Ines Montani

Lo primero que debo decir sobre el tutorial de Ines, es que la documentación es excelente. Siempre que tuve algún tipo de duda al respecto, encontré una posible solución leyendolo.

Paso 1: Los ingredientes.

Estos cursos poseen: (1) filminas con texto, (2) ejercicios de opción múltiple y (3) ejercios de completar parte de código faltante.

El tutorial se divide en capítulos. Cada uno de estos tres “bloques” en distinta combinación puede constituir un capítulo.

Lo que deberías realizar es adaptar tu material antes de seguir avanzando con el tutorial o generarlo directamente considerando este formato.

Paso 2: Preparación.

Paso 3: La cocción.

Para que el curso funcioné se realiza una coccion desde la branch de github desde la página de “Mybinder.org”. Esto permite generar una máquina virtual que permite que corra el código que cargamos en el curso.

Este pasó, como toda cocción, puede demorar algún tiempo.

Paso 4: Decoración

Los cambios más significativos constan de (1) agregar un logo y (2) modificar los colores de los botones.
